Security Architecture

Agent wallets are system-generated, never touch your main wallet

Dedicated Agent Wallet

System generates a separate wallet per user. Your browser wallet is never used for execution.

Encrypted Private Keys

AES-256-CBC + scrypt with random per-key salt. Keys decrypted only at execution time.

Slippage Protection

Every trade validates slippage against user-set max. Blocks execution if exceeded.

One-click toggle in the header. AI reviews every trade before it executes.
